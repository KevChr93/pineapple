Pineapple Payments

is an enterprise level transaction broker software that assists in making payments to any utility company providing gas, water and sewerage, electricity, heating and other utility services of any size. Pineapple Payments allows utility companies to input their data and their customer's data throught our client self-service portal via a gateway. Bills can be sent out via email. The billing module can be used both as standalone module and integrated with an existing system. Servers are hosted by Amazon, so it has reliable power supply and excellent internet connection, additionally regular server maintenances and data backups are performed.
